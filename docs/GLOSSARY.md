# Glossary — Open 3D Store (AXIS)

> Ubiquitous language for this project. Agents MUST use these terms exactly.

| Term                 | Definition                                                     |
| -------------------- | -------------------------------------------------------------- |
| **AXIS**             | Brand name of the 3D model marketplace                         |
| **Model**            | A downloadable 3D print file product (STL, 3MF, OBJ, etc.)     |
| **Creator / Author** | User who publishes models for sale                             |
| **Category**         | Top-level browse grouping (e.g. Miniatures, Mechanics)         |
| **Cart**             | Session list of models selected for purchase                   |
| **Favorites**        | Saved models for later                                         |
| **License**          | Usage rights attached to a model (Commercial, Personal, CC-BY) |
| **Format**           | File type included in a model listing (STL, 3MF, STEP, etc.)   |
| **Badge**            | Listing label: New, Sale, or Free                              |
| **Difficulty**       | Print complexity: Beginner, Intermediate, Advanced             |

## File formats

| Format | Use                                       |
| ------ | ----------------------------------------- |
| STL    | Mesh geometry — universal FDM/resin       |
| 3MF    | Modern archive with metadata and supports |
| OBJ    | Mesh with optional materials              |
| STEP   | CAD source for mechanical parts           |

## Out of scope (current UI)

- Real payment processing
- User authentication backend
- Live 3D viewer (placeholder UI only)
